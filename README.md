# Git Commands Every Developer Should Know

## Basic
- git clone
- git pull
- git fetch
- git push
- git checkout
- git status
- git log
- git add

## Advance
- git rebase
- git merge
- git stash
- git cherry-pick
- git squash

## Clean-up
- git revert
- git reset
  - mixed
  - hard
  - soft
- git clean

## Frequently used combo
- git pull --rebase origin master
- git rebase -i HEAD~3
- git branch -m new-branch-name
- git push -u origin new-branch
